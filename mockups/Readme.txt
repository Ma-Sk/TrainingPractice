В папке authorized авторизированная версия сайта
В unauthorized еавторизированная
В обоих папках лежит html-код прототипа сайта и скриншоты страниц сайта.
Html-код нужен лишь для примерного представления как переходить по ссылкам и НЕ является изначальной версией сайта(Я не собираюсь использовать этот код в своей рабте)
Так что лучше смотреть на скриншоты по причине неадекватного отображения html версии
В unauthorized версии кнопки которых должно не быть будут выводить сообщение об ошибке с предложением авторизироватся.
Слово Save Changes в правом верхнем углу это от редактора прототипов
